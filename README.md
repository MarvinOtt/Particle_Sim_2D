# Particle Sim 2D

__Project creation date:__ 02.03.2017

## Description
This project is a simulation of different kinds of particles.
The physics are normal elastic ball physics.
Types of particles:
* Water
* Steam
* Fire
* Explosive
* Solid

## Technologies
* C#
* MonoGame
